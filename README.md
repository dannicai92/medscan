# MedScan S — 零操作智能药箱盖

&gt; **MedScan S 不是一个需要学习的新工具，而是对现有药箱使用习惯的最小侵入式升级。**  
&gt; 把普通药箱盖换成智能药箱盖，所有行为不变，安全检查自动发生。每次取药，系统自动识别、更新库存、推送至家庭信息中心——**用户零操作，家人全掌握**。

---

## ✨ 核心价值

| 维度 | 普通药箱 | MedScan S |
|---|---|---|
| **取药方式** | 打开药箱盖，取药 | 打开药箱盖，取药（完全相同） |
| **有效期检查** | 需要找眼镜、看清小字、记住结果 | 开盖自动扫描，指示灯 1 秒亮起 |
| **老花眼适配** | ❌ 无法看清小字 | ✅ 只需看灯的颜色 |
| **家人获知情况** | ❌ 无法知晓 | ✅ 推送至家庭信息中心，子女实时掌握 |
| **库存自动管理** | ❌ 需要人工盘点 | ✅ 重量传感器感知取药，自动更新库存 |
| **操作步骤变化** | — | 零变化，用户无需学习任何新操作 |

## 🛠️ 技术架构

| 模块 | 技术方案 |
|---|---|
| **药品识别** | 摄像头 + DataMatrix 解码 / OCR 有效期识别 |
| **过期提醒** | 本地日期比对 + 指示灯驱动（绿/红） |
| **库存感知** | 重量传感器（取药前后差值计算） |
| **数据同步** | 蓝牙/Wi-Fi → 家庭信息中心/手机 App |
| **供电** | 可充电锂电池 / USB-C |

## 📦 安装与使用

### 硬件组装
1. 将原药箱盖替换为 MedScan S 智能盖（卡扣/磁吸适配主流药箱口径）。
2. 首次使用前，长按底部配对键 3 秒，连接家庭 Wi-Fi。

### App 配对
1. 下载 MedScan S App（iOS / Android）。
2. 扫描药箱盖底部二维码完成绑定。
3. 设置家庭信息中心成员，即可接收推送。
